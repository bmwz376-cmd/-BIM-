# NG事例集

実際に起きた失敗事例から学びます。

## ⚠️ この章の目的

- 同じミスを繰り返さない
- チェックポイントを知る
- リスク管理の重要性を理解する

## 🚨 重大NG事例

### 事例1: 施工図不備のBIMモデル化

**状況**:  
施工図が未完成のままBIMモデルを作成。干渉チェックで「問題なし」と判断。

**結果**:  
現場で大量の干渉が発覚。配管ルートを全て変更。工期2週間遅延。

**原因**:  
- 施工図自体が間違っていた
- BIMチェックで「大丈夫」と過信

**対策**:  
- 施工図の精度を先に確保する
- BIMは施工図の品質以上にはならないことを理解する

### 事例2: スリーブ位置ミス

**状況**:  
BIMチェックをスキップ。スリーブ設置後、梁主筋と干渉していることが判明。

**結果**:  
コンクリート打設後、コア抜きで対応。構造補強が必要に。コスト追加100万円。

**原因**:  
- 「時間がない」とBIMチェックを省略
- 躯体図と設備図を重ねずに確認

**対策**:  
- スリーブはBIMチェック必須
- 「後で直せば良い」は絶対にNG

## 💡 失敗から学ぶ教訓

1. **施工図が先、BIMは後**
2. **時間がないは言い訳にしない**
3. **「たぶん大丈夫」は危険**
4. **現場での手直しはコストが高い**
5. **記録を残し、次に活かす**
