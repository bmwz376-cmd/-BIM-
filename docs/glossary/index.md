# 用語集

## 📚 施工図関連用語

### 躯体（くたい）
**English**: Structure / Structural Frame

建物の骨組み。柱・梁・スラブなど構造を支える部分。

### 納まり（おさまり）
**English**: Detail / Coordination / Interface

部材同士が適切に取り合うこと。干渉なく組み上がる状態。

### 干渉（かんしょう）
**English**: Clash / Interference / Conflict

部材同士がぶつかること。物理的に同じ空間を占有する状態。

### 通り芯（とおりしん）
**English**: Grid Line / Structure Line

建物の基準となる線。X1, Y1のように表記。

### FL（エフエル）
**English**: Floor Level

各階の床仕上げ面の高さ基準。

### GL（ジーエル）
**English**: Ground Level

地盤面の高さ基準。建物外部の基準点。

## 🏗️ BIM関連用語

### BIM
**English**: Building Information Modeling

建物の情報をデジタルモデル化する手法。

### 干渉チェック
**English**: Clash Detection

BIMモデル上で部材の衝突を検出する機能。

### LOD
**English**: Level of Development / Level of Detail

BIMモデルの詳細度レベル。100～500まである。

### IFC
**English**: Industry Foundation Classes

BIMデータの国際標準フォーマット。

## 🔧 工事関連用語

### スリーブ
**English**: Sleeve / Core Hole

躯体を貫通する配管用の穴、または筒状の部材。

### 配筋（はいきん）
**English**: Rebar Arrangement / Reinforcement

鉄筋を配置すること、または配置された状態。

### かぶり厚さ
**English**: Concrete Cover

コンクリート表面から鉄筋までの厚さ。
