# 講師用運用ガイド

## 🎯 教える際の基本姿勢

### 1. 施工図が先、BIMは後
**最重要**: 施工図を理解していない受講者に、いきなりBIMを教えない。

### 2. 5点チェックリストを必ず使う
各レッスンの「5点チェックリスト」を必ず説明：

1. どの工程か
2. 誰の仕事か
3. 何を決めるためか
4. 施工図のどの情報と関係するか
5. やらないと何が起きるか

### 3. NG事例を必ず紹介
失敗から学ぶことが最も効果的。

### 4. 画像を拡大して説明
画像はクリックで拡大できる。詳細を指し示しながら説明。

### 5. 小テストで理解度確認
各レッスン後に必ず小テストを実施。60%未満は復習。

## 📚 推奨教え方

### 初回（1日目）
1. 「施工図が分からないと使えない」を強調
2. 施工図の基礎用語を教える
3. 実際の施工図を見せる

### 2回目以降
- 前回の復習から始める
- 新しいレシピを少しずつ追加
- 実際にソフトを操作させる

## 📝 記録と評価

受講者の進捗を記録：
- 小テストの結果
- 質問内容
- 理解度の判定
