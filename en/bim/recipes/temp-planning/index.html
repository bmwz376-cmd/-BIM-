    仮設計画検討 - BIM活用レシピ¶

    📋 概要¶

    BIMを使って足場・タワークレーン等の仮設設備を可視化し、施工手順と安全性を検証

    • 目的: 仮設計画の可視化、施工動線の確保、安全性の事前検証、近隣への説明
    • 工程: 施工計画段階（着工前）
    • 担当: 元請（施工管理、安全管理）+ 仮設専門会社
    • 所要時間: 1棟 12-20時間

    🔍 前提知識（施工図）¶

    施工図が分からないと使えません

    以下の施工図の知識が必須です：

    • 仮設計画図: 足場、クレーン、仮設事務所などの配置を示した図面
    • 総合施工計画書: 工事全体の施工方法、工程、安全対策を記載した書類
    • 足場: 高所作業のための一時的な作業床（枠組足場、くさび式足場など）
    • 揚重計画: 資材を上げる方法（タワークレーン、移動式クレーン、荷揚げ機）
    • 作業半径: クレーンのブームが届く範囲
    • 動線計画: 作業員、資材、重機の移動経路
    • 近接施工: 隣接建物や道路に近い場所での施工

    📊 図解¶

    図1: BIMモデルでの仮設計画（タワークレーン・足場配置）（出典: 施工BIM④）

    図2: 施工ステップごとの仮設状況シミュレーション（出典: 施工BIM④）

    👨‍🏫 講師が必ず言う5点¶

    ### 1️⃣ 工程 施工計画段階（着工前）、近隣説明会でも活用 ### 2️⃣ 誰の仕事 - **元請（施工管理）**: 全体計画の立案、BIMモデル作成 - **元請（安全管理）**: 安全性の検証、労働基準監督署への届出 - **仮設専門会社**: 足場・クレーンの詳細計画、安全性計算 - **近隣住民**: 工事の影響範囲を理解（説明を受ける） ### 3️⃣ 何を決めるためか - タワークレーン・足場の配置位置 - 作業動線の確保（人・資材・重機） - 近隣建物・道路への影響範囲 - 施工ステップごとの仮設状況 ### 4️⃣ 施工図との関係 - **仮設計画図**: 仮設設備の配置情報を取得 - **躯体図**: 建物と仮設の位置関係を確認 - **総合施工計画書**: 施工手順との整合確認 - **敷地図**: 敷地境界、近隣建物との関係 ### 5️⃣ やらないと起きること - クレーンが届かず資材を上げられない - 足場が不足して作業できない - 近隣建物にクレーンが接触 - 作業動線が確保できず工程遅延 - 安全対策不備による事故

    🚀 実施手順¶

    ステップ1: 敷地・建物モデルの準備¶

    graph LR
    A[敷地境界を設定] --> B[建物モデル配置]
    B --> C[近隣建物をモデル化]
    C --> D[道路・電柱を配置]
    D --> E[仮設設備を追加]
    1. 敷地の3Dモデル化
    2. 敷地境界、高低差
    3. 隣接建物（高さ・距離）

    4. 道路、電柱、架空線

    5. 建物モデルの配置

    6. 施工中の建物を配置
    7. 施工ステップごとのモデル作成

    ステップ2: 仮設設備の配置¶

    **仮設設備の種類**: - **タワークレーン**: 位置、高さ、作業半径、旋回範囲 - **足場**: 種類（枠組・くさび式）、高さ、範囲 - **仮設事務所・倉庫**: 配置位置、搬入路 - **荷揚げ機**: 台数、設置位置 - **仮囲い**: 高さ、位置、出入口

    ステップ3: 干渉・安全性チェック¶

    チェック項目 確認内容 対策
    クレーン旋回範囲 隣接建物・道路に出ないか 位置変更、旋回範囲制限
    足場配置 十分な作業床が確保できるか 足場範囲の拡大、種類変更
    作業動線 人・資材の移動に支障はないか 動線の分離、一方通行化
    揚重能力 資材を全範囲に上げられるか クレーン台数・位置の変更
    近隣影響 騒音・振動・日照の影響範囲 防音シート、作業時間の制限

    ステップ4: 施工ステップのシミュレーション¶

    **4Dシミュレーション**（時間軸を加えた可視化）: 1. **フェーズ1: 基礎工事** - 山留め足場、クレーン設置 2. **フェーズ2: 躯体工事** - 外部足場、タワークレーン本格稼働 3. **フェーズ3: 仕上げ工事** - 足場の段階的解体、揚重機への切替 4. **フェーズ4: 仮設撤去** - クレーン解体、足場完全撤去

    💡 講師ノート¶

    教え方のコツ¶

    1. 「仮設は見えないコスト」を強調
    2. 仮設費は工事費の10-15%を占める

    3. BIMで最適化すればコスト削減できる

    4. 安全性の重要性を繰り返す

    5. クレーン事故は死亡事故につながる

    6. 近隣への損害も甚大（建物破損、道路封鎖）

    7. 実際の事故事例を紹介

    8. クレーン旋回時の隣接建物接触事故

    9. 足場からの墜落事故

    10. 近隣説明での活用を説明

    11. 2D図面では住民に理解されにくい
    12. 3Dアニメーションなら「いつ、どこに、何がある」が明確

    よくある質問と回答例¶

    Q: 仮設計画のBIMは必須ですか？

    A: 法律上の義務ではありませんが、複雑な現場や近隣への影響が大きい現場では非常に有効です。安全性向上とコスト削減が期待できます。

    Q: タワークレーンの位置はどうやって決めますか？

    A: ①全範囲に資材を上げられる ②隣接建物に旋回範囲が出ない ③基礎に設置スペースがある、の条件を満たす位置を探します。BIMで試行錯誤できます。

    Q: 4Dシミュレーションは誰が見るのですか？

    A: 施工管理者（工程確認）、安全管理者（安全性検証）、近隣住民（工事説明）、発注者（進捗報告）など、様々な関係者が活用します。

    ❓ 小テスト¶

    ### Q1: 仮設計画のBIM検討を実施する最適なタイミングは？ - [x] A. 施工計画段階（着工前） - [ ] B. 基礎工事開始後 - [ ] C. 躯体工事完了後 - [ ] D. 仕上げ工事開始前 **解説**: 着工前の施工計画段階で実施し、仮設設備の発注や近隣説明に活用します。 ### Q2: タワークレーンの配置で最も重要な確認事項は？ - [ ] A. クレーンの色 - [x] B. 旋回範囲が敷地外に出ないか - [ ] C. クレーンの製造メーカー - [ ] D. クレーンの価格 **解説**: 旋回範囲が敷地外（特に道路や隣接建物）に出ると、法律違反や事故の原因になります。 ### Q3: 4Dシミュレーションの「4D」とは何を意味する？ - [ ] A. 4つの建物 - [ ] B. 4方向から見る - [x] C. 3次元（3D）+ 時間軸 - [ ] D. 4段階の工事 **解説**: 3次元の空間情報に時間軸を加え、施工ステップごとの状況を可視化するのが4Dシミュレーションです。 ### Q4: 仮設計画のBIMモデルで近隣説明に最も効果的なのは？ - [ ] A. 詳細な数値データ - [x] B. 3Dアニメーションでの工事進行の可視化 - [ ] C. 専門用語を多用した説明 - [ ] D. 工事費の明細 **解説**: 3Dアニメーションは専門知識がない近隣住民にも理解しやすく、工事の影響範囲を明確に伝えられます。 ### Q5: 仮設費が工事費全体に占める割合の目安は？ - [ ] A. 1-3% - [x] B. 10-15% - [ ] C. 30-40% - [ ] D. 50%以上 **解説**: 仮設費は工事費の10-15%程度を占め、最適化によるコスト削減効果が大きい項目です。
